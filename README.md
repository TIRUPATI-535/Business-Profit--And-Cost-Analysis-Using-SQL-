# 📊 Business Profit & Cost Analysis Using SQL

## 📌 Project Overview

The **Business Profit & Cost Analysis** project is a SQL-based business analytics project designed to analyze sales performance, product profitability, and customer purchasing behavior. The project uses relational database concepts and SQL queries to generate meaningful business insights that help improve profitability and support data-driven decision making.

## 🔍 Key Queries

### Total Revenue

```sql
SELECT
SUM(p.selling_price * s.quantity) AS Total_Revenue
FROM Products p
JOIN Sales s
ON p.product_id = s.product_id;
```

### Total Profit

```sql
SELECT
SUM((p.selling_price - p.cost_price) * s.quantity)
AS Total_Profit
FROM Products p
JOIN Sales s
ON p.product_id = s.product_id;
```

### Top Selling Products

```sql
SELECT
p.product_name,
SUM(s.quantity) AS Total_Quantity_Sold
FROM Products p
JOIN Sales s
ON p.product_id = s.product_id
GROUP BY p.product_name
ORDER BY Total_Quantity_Sold DESC;
```
